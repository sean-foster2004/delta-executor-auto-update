## Overview

Delta Executor is a Windows-oriented Roblox script executor made for running Lua scripts from a streamlined desktop interface. It is centered on quick script access, an included script hub, and a workflow that remains straightforward for daily use on Windows 10 and Windows 11.

The project emphasizes a small footprint, auto-update support, and features that help manage execution work, including persistent queues and batch runs. Multi-language UI support is also included, which helps the app feel more usable across different locales while working with Roblox scripts and script hub content.
